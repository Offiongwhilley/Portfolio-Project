GUESSING GAME

The player is expected to guess random numbers within a range of numbers.

Home Play Game

These game has 3 key features to wit:

  • A place to input your answers;
  • A feature that reads your scores as you play;
  • A feature that allows you play again.

About the Game

The reason I came up with this fun and exciting game is to let the player work on his/her skills in guessing a random number with only having a limited knowledge and hints as to what the exact mystery number is.
